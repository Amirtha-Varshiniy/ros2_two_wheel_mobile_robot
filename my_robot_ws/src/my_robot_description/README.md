# ROS2 Two_Wheel Mobile Robot
A simple ROS2 robot description package built using URDF and Xacro

## Features
- Two-wheel differential drive robot
- Caster wheel
- Modular Xacro files
- RViz visualization
- Robot State Publisher
- Joint State Publisher GUI

## ROS Version
ROS2 Jazzy
